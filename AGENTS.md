# Agent Instructions

## Quick Start

For agent environments, use these convenience scripts:

- `./bin/agents/setup.sh` - Complete setup (backend + frontend + Playwright)  
- `./bin/agents/run_playwright.sh` - Run Playwright tests (auto-starts/stops servers)
