# Analytics

Analytics provides operational reporting for Spiff Arena process models.
It helps process owners and operations teams understand which models are active, which are completing, where errors are occurring, and how long work is taking.

Use the standalone [Analytics documentation](https://spiff.works/docs/analytics/) when you want to learn how to use the analytics UI or call the analytics API.

The standalone docs cover:

- analytics UI navigation for process groups and models;
- date and metadata filtering;
- Overview, Active, Complete, and Errors views;
- BPMN diagram overlays for tasks and milestones;
- duration thresholds and color schemes;
