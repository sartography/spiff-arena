# Ed

Ed is a next-generation BPMN editor for teams that want to move from workflow idea to executable process model quickly.
Teams can author BPMN diagrams, build forms, record, play back, and refine workflows in the browser, sync model files with GitHub, and use AI-assisted authoring before deploying those workflows in Spiff Arena.

For setup and authoring guidance, read the standalone [Ed documentation](https://spiff.works/docs/ed/).

To learn more about Ed availability, contact SpiffWorks through [spiff.works](https://spiff.works/).
