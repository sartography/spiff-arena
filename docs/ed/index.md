# Ed

Ed is a browser-based BPMN authoring studio for teams that want to move from workflow idea to executable process model quickly.
Authors can design diagrams, build JSON Schema forms, run and debug processes in the browser, synchronize model files with GitHub, and use AI assistance to refine workflows before they are deployed in SpiffArena.

For setup and authoring guidance, read the standalone [Ed documentation](https://spiff-ed.readthedocs.io/en/latest/).

To learn more about Ed availability, contact SpiffWorks through [spiffworkflow.org](https://www.spiffworkflow.org/).
