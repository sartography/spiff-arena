# Extend the System

SpiffArena provides multiple powerful mechanisms to extend and customize the system to meet your specific business requirements. This section covers the various ways you can enhance SpiffArena's capabilities.

:maxdepth: 1

## Overview

SpiffArena offers three extension methods: **Extensions** for custom frontend UI and pages, **Global Scripts** for reusable backend logic and calculations, and **Connectors** for external system integrations. These methods can be combined to create comprehensive custom solutions for your organization.
