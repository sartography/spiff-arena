# SpiffExample

SpiffExample is a sample implementation of SpiffWorkflow, the Python workflow engine that powers BPMN execution in SpiffArena.
Use the standalone [SpiffExample documentation](https://spiffexample.readthedocs.io/en/latest/index.html) when you want to learn how BPMN models become executable workflows with SpiffWorkflow.

SpiffExample focuses on using SpiffWorkflow to create business applications from BPMN documents.
It explains BPMN as a shared process language for business and technical teams, then shows how those diagrams can describe precise execution behavior for an automation engine.

That documentation includes:

- BPMN overview material and simple workflow examples.
- Basic concepts, example code, and diagrams for gateways, scripts, multi-instance tasks, data updates, and loops.
- DMN and decision table examples.
- Jinja documentation strings.
- Advanced BPMN features such as lanes, subprocesses, messaging, timer events, boundary events, and call activities.
- Serialization, deserialization, scripting enhancements, and setup guidance for using or extending the example repository.

Use the SpiffArena documentation when you are working with the web application, deployment, permissions, process model management, connector proxies, forms, user tasks, and runtime administration.
Use SpiffExample when you need a BPMN-first learning path for the SpiffWorkflow engine.
